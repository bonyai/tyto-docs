`@bonya-ai/tyto-mcp` is a [Model Context Protocol](https://modelcontextprotocol.io) server for the Tyto API. It lets any MCP client — Claude Desktop, Claude Code, Cursor, and others — create and manage nests, run commands, move files, and work with previews, snapshots, and keepalive holds, all through tool calls instead of raw HTTP.

The server is self-contained: it talks to the Tyto REST API directly and does not depend on the TypeScript or Python SDK.
